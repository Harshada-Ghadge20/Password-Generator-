# Password-Generator-
Random Password Generator using Python

A secure and beginner-friendly Random Password Generator developed using Python. This project generates strong and random passwords containing uppercase letters, lowercase letters, numbers, and special characters to improve password security.

Features

* Generates strong random passwords
* Includes:
    * Uppercase letters
    * Lowercase letters
    * Numbers
    * Special symbols
* User-defined password length
* Input validation
* Error handling for invalid inputs
* Clean and optimized code structure

Technologies Used

* Python 3
* random module
* string module

Concepts Used

* Functions
* Lists
* String manipulation
* Randomization
* Exception handling
* User input handling

Project Purpose

This project was created to practice Python programming concepts and understand how random password generation works in real-world applications.

It also helps improve:

* Logical thinking
* Code optimization
* Secure password generation concepts

How It Works

1. User enters desired password length
2. Program ensures inclusion of:
    * At least one uppercase letter
    * At least one lowercase letter
    * At least one digit
    * At least one special character
3. Remaining characters are generated randomly
4. Password characters are shuffled for better security
5. Final password is displayed
